.CMD / SHELL(sh)
---------------
- dir (Lista o conteúdo) ls
- cd (Acessa pasta)
- md (Cria pasta) mkdir
- rd (Exclui se vazia)
- copy (Copia arquivos)
- move (Movimenta arquivos)
- del (Exclui arquivos)
- cls (Limpar a tela) clear
- arquivo = dados => pastas = diretório = endereço
- arquivos de lote ou scripts (.bat .sh .vbs)
